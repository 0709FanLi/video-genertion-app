## 2️⃣ 文生图模型

### 2.1 火山引擎即梦 4.0 (jimeng_t2i_v40)
- **服务商**: 火山引擎 (Volcengine)
- **模型ID**: `jimeng_t2i_v40`
- **功能**: 文本生成图片
- **特点**:
  - **支持参考图**：最多6张参考图
  - 高质量图像生成
  - 多风格支持
- **API地址**: `https://visual.volcengineapi.com`
- **参数配置**:
  - 参考图数量: 0-6张
  - 图像尺寸: 可配置
  - 风格: 多种风格可选
- **使用场景**: 
  - 默认文生图模型
  - 支持参考图风格迁移

### 2.2 通义万相多图生图 (wan2.5-i2i-preview)
- **服务商**: 阿里云DashScope
- **模型ID**: `wan2.5-i2i-preview`
- **功能**: 多图融合生成新图
- **特点**:
  - **支持参考图**：最多2张参考图
  - 多图融合生成
  - 1-4张图片输出
- **API地址**: `https://dashscope.aliyuncs.com/api/v1`
- **参数配置**:
  - 参考图数量: 1-2张（必需）
  - 输出图片数: 1-4张
- **使用场景**: 
  - 多图融合创作
  - 参考图混合生成

### 2.3 通义千问文生图 (qwen-image-plus)
- **服务商**: 阿里云DashScope
- **模型ID**: `qwen-image-plus`
- **功能**: 纯文本生成图片
- **特点**:
  - **不支持参考图**：纯文本生成
  - 内置提示词智能改写
  - 5种分辨率可选
  - 同步接口，响应快速（约15秒）
  - 图片自动转存OSS永久保存
- **API地址**: `https://dashscope.aliyuncs.com/api/v1`
- **参数配置**:
  - 支持5种分辨率：
    - `1664*928` (16:9)
    - `1472*1140` (4:3)
    - `1328*1328` (1:1，默认)
    - `1140*1472` (3:4)
    - `928*1664` (9:16)
  - 提示词长度：最长800字符
  - 输出数量：固定1张
  - 提示词智能改写：默认开启
  - 水印：默认不添加
- **使用场景**: 
  - 纯文本创意图片生成
  - 快速原型设计
  - 不需要参考图的场景
  - 多种长宽比需求

## 3️⃣ 图生视频模型

### 3.1 火山引擎即梦 - 文生视频 (volc-t2v)
- **服务商**: 火山引擎 (Volcengine)
- **模型ID**: `jimeng_t2v_v30_1080p`
- **功能**: 纯文本生成视频
- **特点**:
  - **无需图片输入**
  - 固定1080P分辨率
  - 可选5s/10s时长
  - 支持6种长宽比
- **参数配置**:
  - 时长: 5秒 / 10秒
  - 分辨率: 1080P（固定）
  - 长宽比: 16:9 / 4:3 / 1:1 / 3:4 / 9:16 / 21:9
- **使用场景**: 
  - 纯文本生成视频
  - 无参考图片的创意视频生成

### 3.2 火山引擎即梦 - 单图首帧 (volc-i2v-first)
- **服务商**: 火山引擎 (Volcengine)
- **模型ID**: `jimeng_i2v_v30_1080p_first`
- **功能**: 图生视频（首帧模式）
- **特点**:
  - **需要1张首帧图片**
  - 固定1080P分辨率
  - 可选5s/10s时长
  - **推荐使用，默认模型**
- **参数配置**:
  - 首帧图片: 必需
  - 时长: 5秒 / 10秒
  - 分辨率: 1080P（固定）
  - 长宽比: 自动适配首帧
- **使用场景**: 
  - 默认图生视频模型
  - 单张图片生成动态视频

### 3.3 火山引擎即梦 - 首尾帧 (volc-i2v-first-tail)
- **服务商**: 火山引擎 (Volcengine)
- **模型ID**: `jimeng_i2v_v30_1080p_first_tail`
- **功能**: 图生视频（首尾帧插值）
- **特点**:
  - **需要2张图片（首帧+尾帧）**
  - 固定1080P分辨率
  - 可选5s/10s时长
  - 帧间插值生成
- **参数配置**:
  - 首帧图片: 必需
  - 尾帧图片: 必需
  - 时长: 5秒 / 10秒
  - 分辨率: 1080P（固定）
  - 长宽比: 自动适配首帧
- **使用场景**: 
  - 首尾帧插值视频生成
  - 精确控制视频起止画面

### 3.4 通义万相 - 极速版 (wanx-kf2v-flash)
- **服务商**: 阿里云DashScope
- **模型ID**: `wan2.2-kf2v-flash`
- **功能**: 首尾帧生成视频（极速版）
- **特点**:
  - **需要1张首帧图片**（尾帧可选）
  - 可选480P/720P/1080P分辨率
  - 固定5秒时长
  - 生成速度快
- **参数配置**:
  - 首帧图片: 必需
  - 尾帧图片: 可选
  - 时长: 5秒（固定）
  - 分辨率: 480P / 720P（默认） / 1080P
  - 长宽比: 自动适配首帧
- **使用场景**: 
  - 快速视频生成
  - 多分辨率选择

### 3.5 通义万相 - 专业版 (wanx2.1-kf2v-plus)
- **服务商**: 阿里云DashScope
- **模型ID**: `wanx2.1-kf2v-plus`
- **功能**: 首尾帧生成视频（专业版）
- **特点**:
  - **需要1张首帧图片**（尾帧可选）
  - 固定720P分辨率
  - 固定5秒时长
  - 支持复杂运动
- **参数配置**:
  - 首帧图片: 必需
  - 尾帧图片: 可选
  - 时长: 5秒（固定）
  - 分辨率: 720P（固定）
  - 长宽比: 自动适配首帧
- **使用场景**: 
  - 复杂运动视频生成
  - 高质量视频输出

## 📝 备注

1. **所有模型均为在线API调用**，无需本地部署
2. **API密钥已配置在`.env`文件**中
3. **支持热切换**：用户可在前端UI中自由切换模型
4. **自动降级**：当某个模型不可用时，自动使用默认模型
5. **OSS集成**：所有生成的图片和视频自动上传到阿里云OSS永久存储
